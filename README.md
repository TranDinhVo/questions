# Question Frontend

Đây là project frontend thuần sử dụng ReactJS và JSON Server để giả lập backend.

## Yêu cầu hệ thống

- Node.js >= 16.x
- npm hoặc yarn

## Hướng dẫn cài đặt và chạy dự án
```sh
1. Clone project về máy

2. Khởi động database
cd database
npm install
npm start

3. Chạy frontend ReactJS
cd question-front-end
npm install
npm start
